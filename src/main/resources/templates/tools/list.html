Bulk Tool Upload
How the Tool Upload Process Works

This feature allows you to efficiently import multiple tools at once using a standardized Excel template:

  1. Download Template: Use the download button below to get the pre-formatted Excel template
  2. Fill Out Data: Enter tool information in the designated columns with the exact headers shown
  3. Upload File: Select your completed Excel file using the upload button
  4. Automatic Processing: The system validates format and creates tool records automatically

Note: The Excel file must contain the exact column headers as specified in the template for successful processing.

Required Excel Format

Your Excel file must have these exact column headers in the first row:

  • ATool Name
  • BTool Name 2
  • CType
  • DModel Number 1
  • EModel Number 2
  • FSerial Number 1
  • GSerial Number 2
  • HLocation
Headers are case-sensitive and must match exactly as shown above.
Location Column Format

The Location column (H) supports flexible matching for the following formats:

Arizona Locations:
  • Arizona Fab 52
  • AZ F52
  • AZ52
  • AZF52
New Mexico Locations:
  • New Mexico Fab 11
  • NM F11
  • NM11
  • NMF11
Ireland Locations:
  • Ireland Fab 24
  • IE F24
  • IE24
  • IEF24
Location matching is case-insensitive and flexible. If no match is found, the default location will be used.

How bulk upload works (summary)
  • Duplicates = Overwrite: Existing tools are updated field-by-field; IDs/relationships stay the same.
  • GasGuard detection: If a row has an Equipment Location (or is marked GasGuard), it is treated as AMATGASGUARD.
  • GasGuard overwrites: We keep the existing Type and do not change Tool Name or Tool Name 2. System, Equipment Location, Config#, Equipment Set, Serials, and Model# may be updated.
  • Location resolution: If Excel Location is blank, the tool is set to your current site (shown with a green arrow unless it equals the existing site, which shows red as no-op).
  • Equipment Set: Values like 100 and 100% are treated the same.
  • Status/Set Date: New tools default to NOT_STARTED with Set Date = today.
  • New tools: Non-duplicates create a new tool. GasGuard names use System if present, else Equipment Location.
